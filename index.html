📡 API Endpoints

GET /health
Health check - Verify API is running
POST /api/students
Create a new student record
{ "name": "Alice Brown", "department": "Computer Science", "cgpa": 3.75, "email": "alice@university.edu" }
GET /api/students
Get all students with filtering, searching, sorting, and pagination
?department=Computer%20Science&min_cgpa=3.5&sort_by=cgpa&sort_order=desc&page=1&per_page=10
GET /api/students/{id}
Get a single student by ID
PUT /api/students/{id}
Update a student record (partial update supported)
DELETE /api/students/{id}
Delete a student record
GET /api/students/stats/summary
Get student statistics and analytics

⚙️ Technology Stack

  • Flask 2.3.2
  • SQLAlchemy 2.0.19
  • SQLite Database
  • Python 3.7+
  • RESTful API
  • JSON responses

🚀 Quick Setup

# Clone repository git clone https://github.com/anikethsaig3-hash/flask-student-api.git # Create virtual environment python -m venv venv source venv/bin/activate # Install dependencies pip install -r requirements.txt # Seed database python seed.py # Run API python app.py

📋 Student Model

  • id - Auto-increment
  • name - String (1-100)
  • department - String (1-50)
  • cgpa - Float (0.0-4.0)
  • email - Unique email
  • timestamps - Created & updated

🧪 Testing

# Run all tests python -m unittest test_api.py -v # Run specific test python -m unittest test_api.StudentAPITestCase.test_create_student_success -v # Run with coverage coverage run -m unittest test_api.py coverage report

✓ Validation Rules

  • Name: 1-100 characters
  • Department: 1-50 characters
  • CGPA: 0.0 to 4.0
  • Email: Valid format & unique
  • Auto-sanitization
  • Type checking

📊 HTTP Status Codes

  • 200 - OK/Success
  • 201 - Created
  • 400 - Bad Request
  • 404 - Not Found
  • 409 - Conflict (Duplicate)
  • 500 - Server Error
